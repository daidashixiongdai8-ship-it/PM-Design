睡眠引导 APP

PRD V 1.0.0  ·  2026-05-25  ·  iOS & Android 双端

产品定位

一款主动引导用户放下手机、快速入睡的睡眠辅助工具。不是又一个白噪音播放器 —— 而是帮你"戒掉睡前刷手机"的睡眠引导工具。通过极简的睡前仪式、轻唤醒闹钟、以及睡眠激励庄园，构建从"打开 APP → 放下手机 → 安稳入睡 → 醒来获得正向反馈"的完整闭环。

技术前提

项目方案
平台iOS + Android 双端
账号体系本版本不做
数据存储混合：核心功能本地（SQLite/JSON），天气数据上云
天气 API和风天气免费 API
音频iOS AVAudioEngine / Android ExoPlayer · 29 个音频文件本地打包 · ~120MB
打赏静态图片，无 SDK
开发方式Vibe coding · 单人 · 4 周 MVP

本版本不做
